OwlEyes

Always watching your services

Welcome to OwlEyes Monitoring Service

OwlEyes is a comprehensive monitoring solution that helps you keep track of your projects and services in real-time. Create projects, set up monitors, and never miss a downtime again.

API Documentation GraphQL Interface GitHub Repository

Features

Project Management

Create and organize multiple projects with custom labels, descriptions, and tags.

Ping Monitoring

Check if your servers are up by establishing TCP connections to specific hosts and ports.

Website Monitoring

Monitor websites for availability, HTTP status codes, and specific content keywords.

Status Badges

Add status badges to your README or website to show real-time monitoring status.

Multiple Views

View monitor history in list, calendar, or graph mode with real-time updates.

API Access

Access all functionality through our RESTful API or GraphQL interface.

Getting Started

Clone the repository and run with Docker Compose:

git clone https://github.com/yourusername/owleyes.git
cd owleyes
docker-compose up

Then access the application at http://localhost:8000 and the frontend at http://localhost:3000.
